# Pipeline de Dados Quantitativos: Impacto Macroeconômico no Setor Bancário (ITUB4)

Este projeto constrói um pipeline de dados ponta a ponta (End-to-End) para extrair, armazenar, modelar e analisar o impacto histórico da Taxa Selic sobre as ações do Itaú Unibanco (ITUB4).

## 🛠️ Arquitetura e Tecnologias
- **Linguagem:** Python 3.x
- **Ingestão:** APIs do Banco Central (SGS) e Yahoo Finance
- **Armazenamento:** SQLite
- **Transformação:** SQL Avançado (Próximo Passo)

## 📈 Status do Progresso
- [x] **Passo 1:** Ingestão automatizada de dados brutos e staging no SQLite.
- [ ] **Passo 2:** Modelagem de Dados com SQL.
- [ ] **Passo 3:** Análise Estatística e Visualização (Python).
- [ ] **Passo 4:** Relatório de Insights e Documentação Final.
